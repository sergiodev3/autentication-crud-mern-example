# MERN Full Stack Auth + CRUD - Guia para estudiantes

Este repositorio te ensena a construir una aplicacion MERN completa paso a paso, desde backend y frontend hasta seguridad, testing y deploy.

La idea no es solo clonar y correr, sino que puedas escribir el codigo mientras aprendes la logica de cada parte.

## Estructura de aprendizaje

1. `step_1`: Backend API (Node.js + Express + MongoDB + JWT + CRUD).
2. `step_2`: Frontend (React + Vite + Router + Axios + dashboard).
3. `step_3`: Seguridad, testing, CI y deploy (Railway + Vercel).

## Como seguir los steps

1. Empieza por el README del `step_1`.
2. Completa ese paso y valida que todo funcione.
3. Continua con `step_2` y conecta frontend con el backend.
4. Termina en `step_3` para dejar el proyecto listo para produccion.

Cada step incluye instrucciones especificas, comandos, dependencias y flujo recomendado.

## Requisitos previos

- Node.js 20+
- npm 10+
- MongoDB local o MongoDB Atlas
- Git

## Resultado final esperado

Al completar los 3 steps tendras:

1. API REST con autenticacion JWT y CRUD protegido.
2. Frontend React con login, rutas privadas y gestion de productos.
3. Seguridad base, pruebas automatizadas, CI y deploy en nube.
